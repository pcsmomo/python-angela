## PyCharm Shortcut

⇧ ⌃ ⌥ ⌘ ⬆️ ⬇️ ⬅️ ➡️

- ⌥ ⌘ L : Reformat Code
- ⌃ ⌥ R : Run
- ⌃ R : Run recent execution
- Fn ⬆️ : Go to the top
- ⌃ G or ⇧ ⌃ G : Select/unselect the next occurrence
